# 🚀 BikinLaris - Jasa Digital

BikinLaris adalah sebuah platform layanan web untuk jasa digital yang dirancang untuk membantu memajukan dan mengoptimalkan bisnis secara online. Proyek ini berfokus pada penyediaan informasi layanan digital yang profesional, responsif, dan mudah diakses oleh calon klien.

🌍 **Live Website:** [BikinLaris Web](https://bikinlaris.netlify.app/)

---

## 🛠️ Teknologi yang Digunakan

Proyek web front-end ini dibangun menggunakan teknologi dasar yang ringan dan cepat:
* **HTML5**: Sebagai kerangka dan struktur utama halaman web.
* **CSS3**: Untuk *styling*, tata letak, dan memastikan antarmuka terlihat modern serta responsif.
* **Vanilla JavaScript**: Untuk menambahkan fungsionalitas dan interaktivitas pada halaman.
